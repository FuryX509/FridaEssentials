BiSSL is a Frida-based script used to bypass simple SSL pinning implementations that rely on okhttp3.CertificatePinner in Android applications.
CTF is a Frida script used for runtime bypassing of security logic, behavioral tampering, and diagnostic observation during reverse engineering and security testing of Android applications.
Mhook is a simple Frida-based prototype script embedded with binary-to-string functionality to assist with behavioral tampering and runtime analysis.
cryptoCalls is a general-purpose Frida script for analyzing most common cryptographic libraries used in Android applications. It was originally written by secretdiary.ninja
dex_dump is a Frida-based script used to extract DEX files at runtime. It can assist in bypassing certain packing or obfuscation mechanisms during dynamic analysis of Android applications. This script was originally developed by a third party and is used here for research and security testing purposes.
webview is a script used to extract and monitor URLs from WebView-based applications during runtime analysis.
